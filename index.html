Discord bot

Meowbot is a personal Discord bot project built using the discord.py library. It includes a simple pet simulator that uses Discord's embed, reaction, and messaging functionalities to process and display user input. The bot also has a random video generator function that uses the Youtube Data API to randomly select a YouTube video. The selection of videos can be specified according to channel ID. A manager is currently being developed to analyze and organize user messages based on sender, content, and reactions. Errors when calling bot commands are handled accordingly.

Video game

X.B-2513 is a point and click sci-fi adventure game created for Dr. Paul Toprac's BDP 101 Exploring Digital Arts & Media course. The game was built using the Ren'py game engine and Python. Various game mechanics such as puzzles and branching dialogue are featured in this project. Puzzles include replicating a given pattern of sound by comparing user input, moving obstacles using a small map and simple left/right/up/down controls, and clicking on objects hidden in the background. Developed by a small team to write code, story, and create art and sound assets. Team communicated using Trello and Discord.
